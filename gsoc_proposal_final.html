1. Project Title

Enhancing OWASP FinBot CTF with Multi-Agent Security Simulations and Goal Manipulation Guardrails

2. Project Abstract

The OWASP FinBot CTF aims to expose vulnerabilities in agentic AI systems. My proposal focuses on expanding this ecosystem by building a sophisticated multi-agent simulation using LangGraph. This approach simulates complex interactions between financial agents (e.g., Accountant, Auditor, Approver) to identify edge cases where cross-agent goal manipulation or hijacked reasoning can lead to unauthorized invoice processing. By integrating state-of-the-art prompt injection detection and rule-based guardrails, I will provide a robust sandbox for ethical hackers and developers to learn agentic security.

3. Detailed Project Description

Problem

Agentic AI systems, especially multi-agent swarms, are susceptible to "Goal Manipulation" where an attacker can redirect the swarm's objective through indirect communication or shared memory.

Proposed Solution

I have already developed a Proof of Concept (PoC) demonstrating a secure 3-agent LangGraph workflow. The proposed full project will expand this to:

  • Multi-Agent Architecture: Implement a tiered agent system (Auditor, Processor, Finalizer) using LangGraph for stateful multi-turn reasoning.
  • Challenge Tiers: Create "Capture The Flag" levels where users must bypass or secure the system against Indirect Prompt Injection, Goal Hijacking, and Data Leakage.
  • Security Layer: Implement Stateful Validation Nodes that use historical context checks and guardrails to detect anomalous agent behavior.

4. Deliverables

  • Challenge Set 1: Multi-agent invoice processing simulation with basic goal manipulation vulnerabilities.
  • Challenge Set 2: Advanced prompt injection scenarios involving cross-agent memory exploitation.
  • Detection Framework: A set of Python-based tools/nodes for LangGraph to detect and mitigate injection attacks.
  • Documentation: Comprehensive tutorial on identifying and securing agentic AI workflows.
