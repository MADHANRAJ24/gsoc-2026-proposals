## 1. OWASP Foundation - FinBot CTF
**Title**: Enhancing OWASP FinBot CTF with Multi-Agent Security Simulations and Goal Manipulation Guardrails

**Project Abstract (Copy this)**:
The OWASP FinBot CTF aims to expose vulnerabilities in agentic AI systems. My proposal focuses on expanding this ecosystem by building a sophisticated multi-agent simulation using LangGraph. This approach simulates complex interactions between financial agents (Accountant, Auditor, Approver) to identify edge cases where cross-agent goal manipulation or hijacked reasoning can lead to unauthorized invoice processing. By integrating state-of-the-art prompt injection detection and rule-based guardrails, I will provide a robust sandbox for ethical hackers and developers to learn agentic security.

**Detailed Description (Copy & Paste from [gsoc_proposal_final.html](file:///c:/Users/matha/OneDrive/Pictures/Desktop/google%20summer/gsoc_proposal_final.html))**:
- Multi-Agent Architecture: Tiered system using LangGraph.
- Challenge Tiers: Capture The Flag levels for Indirect Prompt Injection and Goal Hijacking.
- Security Layer: Stateful Validation Nodes with historical context checks.

## 2. The Mifos Initiative - AI Fraud Detection
**Title**: Developing an AI-Driven Fraud Detection and Real-time Transaction Monitoring System for Mifos X

**Proposal Summary (Copy this)**:
This project implements an open-source, AI-powered fraud detection system for Mifos X using anomaly detection and stateful agentic workflows (LangGraph). It features an AI Fraud Engine (XGBoost/LightGBM) for real-time risk assessment and 'Security Agents' for automated transaction monitoring. Deliverables include a scalable fraud detection web service, integration PoC for Mifos Payment Hub, and an administrative audit dashboard to enhance security in microfinance.
