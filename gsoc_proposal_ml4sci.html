1. Project Title

Agentic AI for Automated Lagrangian-Level Model Extraction from Scientific Literature

2. Project Abstract

Efficiently identifying and extracting fundamental physics models (Lagrangians) from a vast body of scientific literature is a bottleneck in high-energy physics research. My proposal for ML4SCI introduces an Agentic AI workflow using LangGraph to automate this extraction process. By orchestrating specialized agents (Literature-Parser, Physics-Validator, Model-Synthesizer), I will develop a system that can accurately identify Lagrangian density terms, parameters, and symmetries within research papers. This framework will leverage stateful multi-turn reasoning to cross-validate extracted data against known physics constraints, significantly accelerating the integration of new theoretical models into simulation pipelines.

3. Detailed Project Description

Problem

Scientific papers often represent Lagrangians in non-standardized formats or symbolic notation that is difficult for traditional NLP tools to parse accurately. Missing or misinterpreted terms can lead to significant errors in downstream Monte Carlo simulations.

Proposed Solution

I will implement a multi-agent swarm dedicated to scientific discovery:

  • Literature Agent: Utilizes advanced RAG techniques to parse complex PDF structures and extract mathematical notation from LaTeX source code or OCR.
  • Validation Agent: Implements rule-based and LLM-based checks to ensure extracted Lagrangians are physically consistent (e.g., gauge invariance, conservation laws).
  • Synthesis Node: Uses LangGraph to coordinate feedback loops between agents, refining the extracted model until it meets a predefined confidence threshold.

4. Deliverables

  • Agentic Extraction Core: A LangGraph-based workflow for parsing and validating Lagrangian models.
  • HEPSIM Integration: A tool to convert extracted Lagrangians into standardized formats compatible with HEPSIM simulation tools.
  • Benchmark Dataset: A curated set of extracted models from 50+ high-impact physics papers for validation.
  • Technical Documentation: Guide on adapting the agentic framework for other scientific extraction tasks.
